The Sauce

This code was written for a Cloudspokes.com backbone.js contest. The requirements were to create a backbone.js demostration using Beasty Boys songs information and the JQuery Mockjax library. Below is summary of the libraries and solutions I rolled together to meet the requirements.

The following is an outline of the tools and patterns I used to build this project. I have moved on to build larger bb projects, but have found that this was a great starting point and have returned to this code a number of time to refresh my memory.

Backbone.js v0.9.2 - This is a javascript MVC library created by the developers at DocumentCloud. It provides a bare 'bones' setup of model, collection, view and router 'classes'.

JQuery Mockjax library (from appendTo) This is a jquery plugin that provides a prototype level set of features that allow a developer to mockup end point API data and calls. We have found this to be very useful when the structure of an API is not finalized, but front end UI feature are needed. Using a mock setup of APIs can really help speed up the development and testing when started in the browser.

Require.js This was my first real go at getting require.js and the AMD pattern into practice. Getting familiar with the inner working of require was great, as it helped get my code production ready.

Grunt.js This is great library + build tool that makes the special sauce of backbone, require and twitter work well together. It worth the time to explore, as it will help speed up your code and roll outs. It is worth noting that grunt handles a lot of the everyday development and deloyment tasks, the following project makes it backbone friendly.

Backbone for Grunt (grunt-bbb) This library extends the grunt.js task pattern to provide backbone.js specific tools for setting up modules, require.js build tasks and examples. While it took a while to get right, as the documentation assumes a lot for someone new to require.js, this pattern makes dev-to-release work much easier and stable.

Common commands:
bbb release - runs a full lint + jst + requirejs + concat + min + mincss to a code base.
bbb server:release - provides a fast node.js local dev setup of the a projects distribution code.

JQuery CenterInClient This is a jquery plugin that makes it easy to center a div within a container. Thanks to Rick Strahl for a helpful blog. This feature will be removed so that the twitter-bootstrap can handle the modal content.
