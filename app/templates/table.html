Backbone.js

This is an example backbone.js application, built for the Cloudspokes.com code challenge in 2012. This example use Require.js, MockJax, Backbone.js and jQuery. The base boilerplate is a backbone.js example provides by Thomas Davis. Moving from a basic backbone.js to a production ready code base requires some planning and code structure. The require.js project helps with this coding pattern.

  • MockJax - This is a great project for building a project that needs to appears to talk to an API for data. For this demo the MockJax library was required by the collections/beastyboys.js file. The mock API endpoint was defined relative to the collection definition to allow for easy refactoring, in the event this code is ready to go live.
  • Require.js - The entire require.js project is a learning curve in itself. It is not intuitive. The need for it is. Its worth the time to dig into. The intro for the project outlines the development approach and is worth the time to read.
